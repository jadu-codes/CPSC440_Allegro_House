# CPSC440 Allegro House Project

## Description
This project uses Allegro in C++ to draw a recognizable house scene using multiple drawing primitives.

## Primitives Used
- Filled Rectangle
- Rectangle
- Filled Triangle
- Line
- Filled Circle
- Filled Ellipse

## Features
- House with roof
- Windows and door
- Sun
- Grass
- Cloud

## How to Run
1. Open the project in Visual Studio.
2. Build the project.
3. Run using Local Windows Debugger.
